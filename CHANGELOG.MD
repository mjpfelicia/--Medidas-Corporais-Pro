# Features 
- no formulário de  medições adicionar as seguintes cadastros a seguir:
    - Altura
    - Cintura

- e calcular as informações a seguir:
    - Gordura Corporal(IMC)
    - Massa Muscular

------------
* quando fizer o cadastro de medições as informações devem ser salva no localstore
* os dados de progresso dever ser carregado a parti do localstorage
* os dados exibidos na home devem ser carregados do localstore, usando o ultimo cadastro de medições e progresso
* o dados de Redução de Gordura Corporal e Ganho de Massa Muscular na home deve usar o histórico de *progresso
* o serviço de acesso ao localstore deve de desacoplado e reutilizado
